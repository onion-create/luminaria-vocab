# Luminaria · 浮光词集

A serene floating vocabulary app with an FSRS adaptive memory algorithm and a signature "floating-light" visual identity.

## Features
- 🧠 FSRS adaptive memory algorithm
- ✨ Floating-light Canvas particle aesthetic
- 📊 Visualized FSRS memory state with customizable colors
- 🌍 26,000+ words across 8 banks (CET-4/6, TOEFL, GRE, etc.)
- 🌓 Bilingual UI (English / 中文)
- 📱 PWA — install as native app, offline-ready

## Live Demo
https://onion-create.github.io/luminaria-vocab/

## Tech
- Single-file HTML (~830KB), zero dependencies
- Canvas 2D rendering with radial gradients
- Glassmorphism + Wes Anderson-inspired palettes
- Service Worker for offline caching
